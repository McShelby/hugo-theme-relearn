Taxonomy / Term Pages

Taxonomy Parameter

Front Matter Th' list o' taxonomies be displayed us'n th' children shortcode. By default th' output be o' type=group but ye can overwrite th' children parameter 'n yer front matter.

​
+++
[params]
  [params.children]
    type = 'card'
+++
---
params:
  children:
    type: card
---
{
   "params": {
      "children": {
         "type": "card"
      }
   }
}

For example 'n this docs th' Categories taxonomy does not show a grouped list like 'n th' default (compare t' th' Tags taxonomy) but some neat cards.

Term Parameter

Front Matter Th' list o' terms be displayed us'n th' children shortcode. Ye can overwrite th' children parameter 'n yer front matter. Use a cascade configurat'n on yer taxonomy plank t' inherit parameter down t' every term plank.

​
+++
[[cascade]]
  [cascade.params]
    [cascade.params.children]
      breadcrumb = false
      descript'n = true
+++
---
cascade:
- params:
    children:
      breadcrumb: false
      descript'n: true
---
{
   "cascade": [
      {
         "params": {
            "children": {
               "breadcrumb": false,
               "description": true
            }
         }
      }
   ]
}

For example 'n this docs th' term planks o' th' Categories taxonomy do not show their breadcrumbs like 'n th' default but th' descript'n o' th' plank.
