7.6.0 (2025-04-03)

New

  • New Information was added to the docs about the kind of data the theme stores in the readers browser.

  • New The new tree shortcode displays lists as nicely formatted trees with configurable icon and icon color.

  • New The icon shortcode has learned the style and color options known from other shortcodes.

  • New The theme supports versioning now, where you can keep older versions of your site accessible via a version switcher at the top of the menu.

  • New The menu configuration for the main menu area, introduced in 7.1.0, is extended to the non-scrolling menu header area between search box and main scrolling area using the sidebarheadermenus option and to the menu footer area using the sidebarfootermenus option.

    The configuration for all sidebar menus is similar and now comes with two new types of menus.

    The divider type adds a horizontal ruler.

    The custom type allows to insert any HTML snippets into the menu. The HTML must be wrapped in a li element to not break HTML validation. The theme ships with the following snippets:

    • languageswitcher: will display the language switcher
    • variantswitcher: will display the variant switcher
    • versionswitcher: will display the version switcher
    • historyclearer: will display a button to clear the history of visited links
